# MAIN (한눈에 보는 웹페이지 그림)

이 문서는 “식문화뱅크 홈페이지”를 **크게 어떤 모양/구성으로 만들지**를 아주 쉽게 설명합니다.  
상세한 규칙(글자 크기, 버튼 동작 등)은 `PRD.md`에 있어요.

## 1) 이 홈페이지는 무엇을 보여주나요?
한 문장으로 말하면, **“식문화뱅크가 어떤 곳인지 소개하고, 연락할 수 있게 해주는 페이지”**예요.

## 2) 페이지는 몇 장(몇 페이지)인가요?
답변 주신 내용 기준으로 **여러 페이지(메뉴별 페이지)**로 만들어요.

- 페이지 예시(초안)
  - `index.html` : 인사말(첫 화면)
  - `about.html` : 식문화뱅크 소개(비전/미션)
  - `services.html` : 주요 사업 영역
  - `contact.html` : 신청·상담(연락처/주소/지도/문의폼)

## 3) 화면 구성(위에서 아래로)
### (1) 맨 위 큰 화면(첫 화면/히어로)
- **배경**: 파스텔 톤(파란색~분홍색 느낌) + 하트 워터마크 느낌
- **오른쪽**: 제공된 이미지(`김미옥 사진-.png`)처럼 인물 사진이 들어갈 수 있음
- **왼쪽 텍스트**: “안녕하세요. 식문화뱅크입니다.” 같은 인사말 요약 + 짧은 소개
- **버튼(선택)**: “상담 신청/문의하기” 버튼을 누르면 아래 “신청 및 상담” 섹션으로 이동

### (2) 소개 섹션
- “식문화뱅크 소개” 문구와 설명
- “비전”, “미션” 문구와 핵심 문장/목록

### (3) 주요 사업 영역 섹션
4개 영역을 카드(박스) 형태로 보기 쉽게 나열
- 식품·영양 교육 멘토
- 식생활 안전 교육
- 영양상품 판매
- 전문가 네트워크 컨설팅

### (4) 신청 및 상담 섹션
연락에 필요한 정보가 한 번에 보이게 정리
- 문의전화 / 이메일 / 주소
- (선택) 지도(카카오맵/네이버지도/구글지도 중 1개)
- 사업자 정보(상호/대표/사업자번호/주소)

### (5) 맨 아래(푸터)
- 간단한 저작권 문구 + 사업자 정보 반복(원하면)

## 4) 사용자가 할 수 있는 행동(버튼을 누르면 어떻게 되나요?)
- 상단 메뉴 클릭 → 해당 페이지로 이동
- 전화/이메일 클릭 → 전화 앱/메일 앱 열림
- (지도 사용 시) 지도 클릭 → 지도 서비스로 이동(네이버지도)

## 5) 우리가 만들 파일(예정)
사용 기술은 **HTML + CSS + JS**만 사용합니다.

- `index.html`: 인사말 페이지
- `about.html`: 소개/비전/미션 페이지
- `services.html`: 주요 사업 영역 페이지
- `contact.html`: 신청·상담/지도/문의 페이지
- `styles.css`: 색/글씨/배치/반응형(모바일) 디자인
- `script.js`: 버튼 클릭 시 스크롤 이동 같은 간단한 동작
- `assets/김미옥 사진-.png`: 제공된 이미지(첫 화면 등에 사용)

> 다음 단계: `PRD.md`에서 “각 페이지에 무엇을 넣을지”를 더 정확히 적습니다.
